# 常见问题

window下安装后:

C:\Users\Administrator>python -V
'python' 不是内部或外部命令，也不是可运行的程序
或批处理文件。

这就是没设好环境变量,解决方法百度

C:\Users\Administrator>pip -V
'pip' 不是内部或外部命令，也不是可运行的程序
或批处理文件。

可能是没设置好环境变量,或者pip压根没装 解决方法重新安装

若是提醒升级可以不必在意

.py文件 鼠标右键中没有 edit with IDLE

解决方法 修改注册表 有3个文件夹(新建)的默认值要修改 具体百度
