# 2，python历史。

宏观上：python2 与 python3 区别： python2 源码不标准，混乱，重复代码太多， python3 统一 标准，去除重复代码。

# 3，python的环境。

编译型：一次性将所有程序编译成二进制文件。 缺点：开发效率低，不能跨平台。 优点：运行速度快。 ：C，C++等等。

解释型：当程序执行时，一行一行的解释。 优点：开发效率高，可以跨平台。 缺点：运行速度慢。 :python ,php,等等。
