# SAI Ver.2的使用

## 默认快捷键

可能因为版本不同有略微差异。

### 画布

Ctrl + Z 撤销

Ctrl + Y 重做

滚轮 缩放倍率

方向键 移动

Ctrl + 方向键 微调移动

---

### 图层

Ctrl + C 复制

Ctrl + P 粘贴

Ctrl + T 剪切

Ctrl + del 填充颜色

F 向下转写（最好修改为空或者其他键防止误触）

Ctrl + U 调整图层HSL

---

### 画笔

[ ] 大小

1-0 浓度

X 切换颜色

CVBN 水彩笔/画笔/喷枪/铅笔

---

### 选区

Ctrl + D 取消选择

shift + 套索 加大选区

alt + 套索 减小选区
