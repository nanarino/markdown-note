##### while。

```python
while 条件:
    循环体
    无限循环。
    终止循环：1，改变条件，使其不成立。
             2,break

continue 跳过本次循环
```
##### while也有else

```python
count = 0
while count <= 5 :
    count += 1
    if count == 3:break
    print("Loop",count)

else:
    print("循环正常执行完啦")
print("-----out of while loop ------")
```
